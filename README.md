# Финальный проект 1 семестра

REST API сервис для загрузки и выгрузки данных о ценах.

## Требования к системе
- go
- psql

Протестировано на следующей версии Go:
```
go version go1.26.0-X:nodwarf5 linux/amd64
```

## Установка и запуск
Чтобы запустить сервер, нужно:
1) Запустить PostgreSQL на порту 5432
2. Установить зависимости и подготовить базу данных при помощи `./scripts/prepare.sh`
3. Запустить сервер скриптом `./scripts/run.sh`


## Тестирование
Для тестирования используется команда `./scripts/tests.sh 1`

### Эндпоинт `POST /api/v0/prices`

Входные данные: zip-архив с данными в теле запроса. Пример - `sample_data.zip`
Выходные данные: JSON-объект, пример:

```
{
  "total_items": 100,
  "total_categories": 15,
  "total_price": 100000
}
```

Описание полей JSON:
- `total_items` — общее количество добавленных элементов.
- `total_categories` — общее количество категорий.
- `total_price` — суммарная стоимость всех объектов в базе данных.

### Эндпоинт `GET /api/v0/prices`

Входные данные: отсутствуют.
Выходные данные: zip-архив с файлом data.csv, в котором находятся данные из базы данных.
